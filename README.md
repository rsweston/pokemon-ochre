# To work on an issue:
* assign yourself to the issue (on the right sidebar) so we know if someone is working on it
* when creating the associated pull request, link the issue so we know they're connected

# To push code:
* make sure you have the latest code pulled to your working branch
* push code to a new branch
* create a pull request to merge that branch into master
* MAKE SURE the pull request is to rweston and NOT rh-hideout!!!!!!!!
